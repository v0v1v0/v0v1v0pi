reduce_single_digit R Documentation

Reduce an Integer, a Date (Time), or a Number (with or without Decimals) to a Single Integer

Description

Takes a character vector coercible to a date using anydate or a date time using anytime; a character vector with numbers; a numeric vector; or an integer vector & computes the sum to a single digit using Mod_octave

The vectors may include periods, dashes, parentheses, colons, and/or spaces. See the examples.

Usage

reduce_single_digit(string)

Arguments

string

character vector coercible to a date using anytime or a date time using anytime; a numeric vector; or an integer vector

Value

a numeric vector with a single digit (integer from 0 - 9)

Author(s)

Irucka Embry
