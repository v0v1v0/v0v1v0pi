test_pkg_dir R Documentation

Test Location is Package Directory

Description

Test whether the working directory is located at the top level directory of the package.

Usage

test_pkg_dir(pkg)

Arguments

pkg

'character' string. Package name.

Value

Formatted file size(s).

Author(s)

J.C. Fisher, U.S. Geological Survey, Idaho Water Science Center

Examples

test_pkg_dir("inldata")
