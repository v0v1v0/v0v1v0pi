rMU R Documentation

Greek Letter mu Shaped Bivariate Data Generation

Description

A function to generate bivariate data with the scatterplot resembling the greek letter mu.

Usage

rMU(n)

Arguments

n

the sample size.

Value

A n times 2 matrix

Author(s)

Klaus Nordhausen, Joni Virta

Examples

x <- rMU(1000)

plot(x)
