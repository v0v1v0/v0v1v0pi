Wheat_IBCF R Documentation

Wheat Data

Description

The package includes a data set based on a portion of the data used in the study of Montesinos-Lopez, O. A.; Montesinos-Lopez, A.; Crossa, J.; Toledo, F. H.; Montesinos-Lopez, J. C.; Singh, P. & Salinas-Ruiz, J. (2017). A Bayesian Poisson-lognormal Model for Count Data for Multiple-Trait Multiple-Environment Genomic-Enabled Prediction. G3: Genes|Genomes|Genetics 7(5):1595â€“1606. http://doi.org/10.1534/g3.117.039974. The data set consists of 250 wheat lines evaluated in 3 environments and 4 distinct traits, i.e. 3000 observations.

Usage

data(Wheat_IBCF)

Format

a TidyData format, 3000 row per 4 columns.

Author(s)

Montesinos-Lopez, O. A.
