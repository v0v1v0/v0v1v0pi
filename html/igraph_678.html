normalize R Documentation

Normalize layout

Description

Scale coordinates of a layout.

Usage

normalize(
  xmin = -1,
  xmax = 1,
  ymin = xmin,
  ymax = xmax,
  zmin = xmin,
  zmax = xmax
)

Arguments

xmin, xmax

Minimum and maximum for x coordinates.
ymin, ymax

Minimum and maximum for y coordinates.
zmin, zmax

Minimum and maximum for z coordinates.

See Also

merge_coords(), layout_().
