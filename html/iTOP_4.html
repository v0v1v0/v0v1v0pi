inner.product R Documentation

Inner product similarity.

Description

Computes the inner product between x and y.

Usage

inner.product(x, y)

Arguments

x

A vector of numbers.
y

A vector of numbers.

Value

The inner product similarity between x and y.

Examples

set.seed(2)
n = 100
x = rnorm(n)
y = rnorm(n)
inner.product(x, y)
