renorm R Documentation

Renormalise image

Description

Pixel data is usually expressed on a 0...255 scale for displaying. This function performs a linear renormalisation to range min...max

Usage

renorm(x, min = 0, max = 255)

Arguments

x

numeric data
min

min of the range
max

max of the range

Author(s)

Simon Barthelme

Examples


renorm(0:10)
renorm(-5:5) #Same as above
