iv-accessors R Documentation

Access the start or end of an interval vector

Description

  • iv_start() accesses the start of an interval vector.

  • iv_end() accesses the end of an interval vector.

Usage

iv_start(x)

iv_end(x)

Arguments

x

⁠[iv]⁠

An interval vector.

Value

The start or end of x.

Examples

x <- new_iv(1, 2)

iv_start(x)
iv_end(x)
