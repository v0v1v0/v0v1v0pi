Military nuclear accidents (Wikipedia)

Description

A table containing the military nuclear accidents.

Usage

military_nuclear_accidents_wiki

Format

A data.table data frame with 76 rows and 4 variables:

Date

Date of the accident

Location

Location of the accident

Type

Type of accident

Description

Description of accident or incident
