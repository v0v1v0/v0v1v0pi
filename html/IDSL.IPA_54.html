SNR xcms

Description

This function calculates S/N values using a method suggested in the xcms paper (Tautenhahn, 2008).

Usage

SNRxcms(int)

Arguments

int

a vector of intensities corresponding to the vector of retention times for the chromatographic peak.

Value

S/N value
