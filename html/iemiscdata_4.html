Civilian nuclear accidents (Wikipedia)

Description

A table containing the civilian nuclear accidents.

Usage

civilian_nuclear_accidents_wiki

Format

A data.table data frame with 28 rows and 5 variables:

Date

Date of the accident

Location

Location of the accident

INES Level

INES level

Type

Type of accident

Description

Description of accident or incident

Source

Wikimedia Foundation, Inc. Wikipedia, 3 September 2023, "List of civilian nuclear accidents", https://en.wikipedia.org/wiki/List_of_civilian_nuclear_accidents.
