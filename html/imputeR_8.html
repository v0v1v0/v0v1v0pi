logistic regression with lasso for imputation

Description

logistic regression with lasso for imputation

Usage

lassoC(x, y)

Arguments

x

predictor matrix
y

response vector

Value

a model object that can be used by the impute function

See Also

cv.glmnet and glmnet

Examples

data(spect)
missdata <- SimIm(spect, 0.1)

impdata <- impute(spect, cFun = "lassoC")
