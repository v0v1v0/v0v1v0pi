selectgd R Documentation

Selecting optimal interaction for GD model.

Description

Selecting optimal interaction for GD model.

Usage

selectgd(formula, data, ndisc, methoddisc)

Arguments

formula

A formula of spatial variables
data

A data frame of dataset
ndisc

A numeric vector of break numbers for respective explanatory variables
methoddisc

A character vector of discretization methods

Value

A list of process and results of optimal interaction for GD model.

Examples

s1 <- selectgd(formula = y ~ xa + xb + xc, data = sim,
               ndisc = c(4,6,6), methoddisc = "quantile")
