is.iteror R Documentation

is.iteror indicates if an object is an iteror.

Description

is.iteror indicates if an object is an iteror.

Usage

is.iteror(x)

Arguments

x

any object.

Value

TRUE if the object has class iteror.

Examples


it <- iteror(1:3)
stopifnot(is.iteror(it))
repeat {
  print(nextOr(it, break))
}
