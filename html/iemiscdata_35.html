gases_table R Documentation

Gaseous Elements and Compounds

Description

A table containing gases at standard conditions. This table is derived from dry_air, gases_plain, and common_gases_simp.

Usage

gases_table

Format

A data.table data frame with 534 rows and 4 variables:

Name

Name of the gas

Formula

Chemical formula of the gas

Molecular weight

Molecular weight of the gas

CAS

CAS number
