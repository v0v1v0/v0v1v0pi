corregirRespuestas R Documentation

Corrección de respuestas

Description

La función corregir respuestas transforma las respuestas de los estudiantes a puntaje. El puntaje puede ser un 1, si la respuesta es correcta, o un 0, si la respuesta es incorrecta. Los valores NA reciben puntaje 0.

Usage

corregirRespuestas(respuestas, clave)

Arguments

respuestas

Un data frame con las alternativas seleccionadas por los estudiantes en cada ítem.
clave

Una data frame con la alternativa correcta para cada ítem.

Value

Un data frame con los aciertos (1) o errores (0) de cada estudiante en cada ítem.

See Also

datos y clave

Examples

respuestas <- datos[, -1]
corregirRespuestas(respuestas, clave)
