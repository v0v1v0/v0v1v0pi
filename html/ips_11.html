beastLog R Documentation

XML Parameter Nodes

Description

A collection of parameter nodes used in the standard template of BEAST2.

Usage

beastLog(l, x, i)

Arguments

l

A character string giving the name (i.e. id or idref attribute) of the desired parameter.
x

A list, internally generated by rbeauti.
i

Integer giving the index to a partition, internally generated by rbeauti.

Value

An object of class XMLNode.
