und_intnet_chicago R Documentation

This data is an intensitynet object containing an undirected network. The base data used is from Chicago, extracted from the spatstat package.

Description

This data is an intensitynet object containing an undirected network. The base data used is from Chicago, extracted from the spatstat package.

Usage

und_intnet_chicago

Format

An object of class intensitynetUnd (inherits from intensitynet) of length 6.
