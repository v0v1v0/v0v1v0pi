Residual Sodium Carbonate (RSC) Calculation

Description

Residual Sodium Carbonate (RSC) Calculation

Usage

RSC(df, convert_to_meq = TRUE)

Arguments

df

Dataframe containing necessary variables (HCO3, CO3, Ca, Mg).
convert_to_meq

Logical, indicating whether to convert concentrations to meq/L (default: TRUE).

Value

RSC value.
