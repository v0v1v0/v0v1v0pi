src_databases R Documentation

List all available databases

Description

Returns a character vector containing the names of all the available databases, in alphabetical order, including the _impala_builtins database.

Usage

src_databases(src, ...)

src_schemas(src, ...)

Arguments

src

object with class class src_impala
...

Optional arguments; currently unused.

Details

src_schemas() is an alias for src_databases()
