hostTaxonomy R Documentation

Cached version of host taxonomy (see vignette for code to build from scratch)

Description

Host taxonomic cached data from NCBI (see vignette for code to re-make)

  • HostSpecies original host name from EDWIP

  • HostTaxID NCBI taxonomic ID

  • HostGenus host genus

  • HostFamily host family

  • HostOrder host order

  • HostClass host class

Usage

data(hostTaxonomy)

Format

A data.frame
