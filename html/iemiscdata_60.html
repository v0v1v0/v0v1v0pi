iemiscdata: Irucka Embry's Miscellaneous Data Collection

Description

iemiscdata provides an eclectic set of data sets [Chemistry, Engineering Economics, Environmental/Water Resources Engineering, Nuclear Accidents, US Presidential Elections, and US Continental Congress Presidents].
