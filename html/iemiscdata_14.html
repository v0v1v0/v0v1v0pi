common_gases_simp R Documentation

Common Gaseous Elements and Compounds Simplified

Description

A table containing common substances that are gases at 25 degrees C and 1.0 atm. This is the simplified table with the elements and compounds combined in a column.

Usage

common_gases_simp

Format

A data.table data frame with 35 rows and 2 variables:

Name

Name of the element or compound

Formula

Molecular formula of the element or compound
