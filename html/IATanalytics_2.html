sampledata R Documentation

Sample Dataset from a Typical IAT

Description

This dataset contains sample data from a typically-structured IAT. This IAT has 220 trials, and this particular dataset does not meet criteria for exclusion.

Usage

data(sampledata)

Format

An object of class tbl_df (inherits from tbl, data.frame) with 220 rows and 6 columns.

Examples

data(sampledata)
