getTagCount R Documentation

Get number of times a tag has been used

Description

getTagCount retrieves a count of the number of times a tag has been used in comments to a picture or video.

Usage

getTagCount(tag, token)

Arguments

tag

Hashtag used to filter media.
token

An OAuth token created with instaOAuth.
