idsa R Documentation

IDSA model with spatial discretization parameters.

Description

IDSA model with spatial discretization parameters.

Usage

idsa(formula, location, data, ndisc, methoddisc,
            methodoverlay = "fuzzyAND")

Arguments

formula

A formula of spatial variables
location

A character vector of location names in a data frame
data

A data frame of dataset
ndisc

A numeric vector of break numbers for respective explanatory variables
methoddisc

A character vector of discretization methods
methodoverlay

A character of spatial overlay methods, including "fuzzyAND" and "intersection"

Value

A list of IDSA results.

Examples

q.fand <- idsa(formula = y ~ xa + xb + xc, location = c("lo", "la"),
               data = sim, ndisc = c(4,6,6), methoddisc = "quantile",
               methodoverlay = "fuzzyAND")
q.ints <- idsa(formula = y ~ xa + xb + xc, location = c("lo", "la"),
               data = sim, ndisc = c(4,6,6), methoddisc = "quantile",
               methodoverlay = "intersection")
