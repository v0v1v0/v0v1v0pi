report_gallery_image R Documentation

Report a gallery image

Description

Report a gallery image as inappropriate.

Usage

report_gallery_image(id, ...)

Arguments

id

An image ID.
...

Other arguments passed to HTTP request functions, for example: token (an OAuth2.0 token) or key (an API key, used by default).

Details

Reports a gallery image as inappropriate.

Value

An object of class “imgur_basic”.
