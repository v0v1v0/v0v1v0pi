clear_token R Documentation

clear_token

Description

Deletes the token from the key ring.

Usage

clear_token(username = getOption("ices.username"))

Arguments

username

the ices username that you require a token for, Default: getOption("ices.username")

Value

no return value

See Also

username key_get

Examples

## Not run: 
clear_token()

## End(Not run)
