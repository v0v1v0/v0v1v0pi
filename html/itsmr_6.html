Forecast using ARAR algorithm

Description

Forecast using ARAR algorithm

Usage

arar(y, h = 10, opt = 2)

Arguments

y

Time series data
h

Steps ahead
opt

Display option (0 silent, 1 tabulate, 2 plot and tabulate)

Value

Returns the following list invisibly.

pred

Predicted values
se

Standard errors
l

Lower bounds (95% confidence interval)
u

Upper bounds

See Also

forecast

Examples

arar(airpass)
