centralization.closeness.tmax R Documentation

Theoretical maximum for closeness centralization

Description

centralization.closeness.tmax() was renamed to centr_clo_tmax() to create a more consistent API.

Usage

centralization.closeness.tmax(
  graph = NULL,
  nodes = 0,
  mode = c("out", "in", "all", "total")
)

Arguments

graph

The input graph. It can also be NULL if nodes is given.
nodes

The number of vertices. This is ignored if the graph is given.
mode

This is the same as the mode argument of closeness(). Ignored if an undirected graph is given.
