Outside interval

Description

Decide if a value x falls outside an interval j[1],j[2] that can be open or closed on the left and/or the right. Either a logical vector equal to x, or the actual values are extracted,

Usage

x %][% j

x %)(% j

x %](% j

x %)[% j

x %].[% j

x %).(% j

x %].(% j

x %).[% j

Arguments

x

A vector
j

A range

Value

logical vector of length x, or, values of x outside the range j

Note

Package 'DescTools' provides similar functions

Examples


# Closed interval
5%][%c(1,5)
5%].[%c(1,5)

# Open interval
5%)(%c(1,5)
5%).(%c(1,5)

# Half-losed interval left
5%](%c(1,5)
5%].(%c(1,5)

# Half-losed interval right
5%)[%c(1,5)
5%).[%c(1,5)
