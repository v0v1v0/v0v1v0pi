seMRprops R Documentation

Multiple binary response

Description

SE's for multiple binary response

Usage

seMRprops(obj)

Arguments

obj

something that can be turned into a matrix

Value

an ses.moecalc object

Author(s)

Junjie Zeng

Examples

x <- data.frame(
    v1 = rbinom(20, 1, 0.8),
    v2 = rbinom(20, 1, 0.3),
    v3 = rbinom(20, 1, 0.5)
)
seMRprops(x)
