ReadStapleGT96SBS R Documentation

Read a 96-channel spectra (or signature) catalog where rownames are e.g. "A[C>A]T"

Description

The file needs to have the rownames in the first column.

Usage

ReadStapleGT96SBS(
  file,
  ref.genome = NULL,
  region = "unknown",
  catalog.type = "counts",
  abundance = NULL,
  sep = "\t"
)
