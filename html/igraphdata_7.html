kite R Documentation

Krackhardt's kite

Description

Krackhardt's kite is a fictionary social network with ten actors. It is the smallest graph for which the most central actor is different according to the three classic centality measures: degree, closeness and betweenness.

Usage

kite

Format

An undirected igraph graph with graph attributes name, layout, Citation, Author, URL, and vertex attributes label, name and Firstname.

Source

http://www.orgnet.com/sna.html

References

Assessing the Political Landscape: Structure, Cognition, and Power in Organizations. David Krackhardt. Admin. Sci. Quart. 35, 342-369, 1990.
