inegi_stind R Documentation

Student Price Index

Description

Returns the student price index. See http://enelmargen.org/ds/ipe/ for more information.

Usage

inegi_stind(token)

Arguments

token

API token supplied by INEGI

Value

Data.frame

Author(s)

Eduardo Flores

Examples

## Not run: 
token <- "webservice_token"
studentinflation <- inegi_stind(token)
