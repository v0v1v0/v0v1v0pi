Table from Appendix A: Hydrologic Soil Groups (HSGs)

Description

A table containing the hydrologic soil groups.

Usage

hsg

Format

A data.table data frame with 4 rows and 2 variables:

Hydrologic Soil Group (HSG)

Hydrologic soil group

Soil textures

Texture of soil
