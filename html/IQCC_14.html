d2 R Documentation

D2 Constant.

Description

This function is used to calculate the mean of the sample relative range (W statistic).

Usage

d2(n)

Arguments

n

The sample size.

Value

Return the value of d2 for a given sample size n.

Author(s)

Daniela R. Recchia, Emanuel P. Barbosa
