aFiles R Documentation

Get and set aFiles.

Description

Get and set the list of aFiles data.frames in an object of ISA.

Usage

aFiles(x)

aFiles(x) <- value

## S4 method for signature 'ISA'
aFiles(x)

## S4 replacement method for signature 'ISA'
aFiles(x) <- value

Arguments

x

An object of class ISA.
value

A list of data.frames containing the assay files. In each data.frame at least the following columns are present: ‚Äù.

Value

A list of data.frames containing the assay files.

The updated object of class ISA.

Examples

## Read example Atwell data set.
isaObject1 <- readISATab(path = file.path(system.file("extdata/Atwell",
                                          package = "isatabr")))

## Extract list of aFiles data.frames.
aFiles(isaObject1)
