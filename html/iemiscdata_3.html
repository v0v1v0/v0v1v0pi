cell_salts_atoms_final R Documentation

Total Number of Atoms of Elements From the 27 Human Cell Salts Table

Description

A table containing the total number of atoms of the 24 chemical elements in the 27 inorganic substances, known as cell salts, found in Human Beings.

Usage

cell_salts_atoms_final

Format

A data.table data frame with 24 rows and 3 variables:

Element Name

The chemical element name

Element Symbol

The chemical element symbol

Number of Atoms

The total number of atoms contained within all 27 cell salts for this particular element
