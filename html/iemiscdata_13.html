common_gases R Documentation

Common Gaseous Elements and Compounds

Description

A table containing common substances that are gases at 25 degrees C and 1.0 atm. The elements and compounds are in separate columns.

Usage

common_gases

Format

A data.table data frame with 23 rows and 4 variables:

Element_Formula

Molecular formula of the element

Element_Name

Name of the element

Compound_Formula

Molecular formula of the compound

Compound_Name

Name of the compound
