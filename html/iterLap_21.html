resample R Documentation

Residual resampling

Description

Perform residual resampling to the result of importance sampling

Usage

resample(n, obj)

Arguments

n

Number of resamples to draw

obj

An object of class IS, as produced by the IS function

Value

Matrix with resampled values

Author(s)

Bjoern Bornkamp
