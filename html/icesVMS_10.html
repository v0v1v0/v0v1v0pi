get_fo_effort R Documentation

Download fishing effort summaries

Description

Download a data.frame of kw fishing hours by country and year for a given ICES ecoregion.

Usage

get_fo_effort(ecoregion)

Arguments

ecoregion

ICES ecoregion

Value

a data.frame
