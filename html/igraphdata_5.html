immuno R Documentation

Immunoglobulin interaction network

Description

The undirected and connected network of interactions in the immunoglobulin protein. It is made up of 1316 vertices representing amino-acids and an edge is drawn between two amino-acids if the shortest distance between their C_alpha atoms is smaller than the threshold value \theta=8 Angstrom.

Usage

immuno

Format

An undirected igraph graph object.

Graph attributes: ‘name’, ‘Citation’, ‘Author’.

Source

See reference below.
