Function to check the format of a numeric column

Usage

Arguments

data frame
nrcode

non response code corresponding to the column
minval

minimum value allowed
maxval

maximum value allowed

Value

0, if success error, if failure

Examples

test_data_numeric("age", data.frame(
  "Age" =  c(21, 15),
  "Name"  = c("John", "Dora")
