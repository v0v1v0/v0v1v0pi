replace.one R Documentation

replace one

Description

Do find and replace for one element of an inner documentation list on 1 Rd file.

Usage

replace.one(torep, REP, 
    txt, verbose = FALSE)

Arguments

torep

tag to find.
REP

contents of tag to put inside.
txt

text in which to search.
verbose

cat messages?
