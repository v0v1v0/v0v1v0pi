call_data_ibb R Documentation

Fundamental Call Function

Description

Fundamental call function of (almost) all IBB Open Data Portal (exceptions apply)

Usage

call_data_ibb(
  call_phrase = "",
  root_url = "https://data.ibb.gov.tr/api/3/action/"
)

Arguments

call_phrase

Suffix of the call url.
root_url

Root of the calls.
