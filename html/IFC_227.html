specialr R Documentation

Special Character Replacement

Description

Helper to replace special character.

Usage

specialr(
  string = "",
  replacement = "_",
  specials = "[\\|\\/|\\:|\\*|\\?|\\\"|'|\\<|\\>|\\|]"
)

Arguments

string

string where specials will be replaced if found.
replacement

string replacement. Default is "_".
specials

Default is '[\|\/|\:|\*|\?|\"|\'|\<|\>|\|]'.
